# Unreal Multiplay Bulls and Cows
Dedicated Server로 구현한 멀티플레이 숫자야구 게임입니다.

## 채팅
게임 내부 채팅을 구현했습니다.

접속한 다른 클라이언트들과 채팅으로 소통해보세요.

가장 먼저 접속한 클라이언트 순서대로 Player1, Player2 형식의 플레이어 이름을 부여받습니다.

## 숫자야구 게임
중요 로직은 게임모드, 서버에서 동작합니다.

### 방장
가장 먼저 서버에 접속한 플레이어는 방장으로 간주됩니다.

방장은 /start, /restart 명령어를 이용해 게임을 시작할 수 있습니다.

### 기회
채팅으로 세 자리의 숫자를 입력해 숫자야구에 참여할 수 있습니다.

게임을 시작하면 모든 플레이어들은 정답 맞추기를 시도할 수 있는 세 번의 기회를 얻습니다.

### 승리
만약 어느 플레이어가 숫자를 맞추는데 성공하면 해당 플레이어의 승리를 출력하고 게임은 종료됩니다.

### 제한시간
게임을 시작하면 30초의 제한시간이 시작됩니다.
30초 내에 숫자를 맞추지 못하거나, 모든 플레이어가 기회를 전부 소모하면 무승부가 됩니다.
